title: "Get Results"
description: "Retrieve scraped Bien'Ici real estate listings"
api: "GET https://api.lobstr.io/v1/results"
---

Retrieve scraped Bien'Ici real estate listings including comprehensive property details, pricing, location, amenities, energy performance ratings, and agency information.

## Headers

<ParamField header="Authorization" type="string" required>
  Your API authentication token. Value: `Token YOUR_API_KEY`
</ParamField>

## Query Parameters

<ParamField query="squid" type="string" required>
  Hash of the squid to get results from. Example: `53317cbe332e408c5c46372a18e4fc76`
</ParamField>

<ParamField query="run" type="string">
  Hash of a specific run (optional). Example: `a1b2c3d4e5f6789012345678901234ab`
</ParamField>

<ParamField query="page" type="integer">
  Page number (default: 1). Example: `1`
</ParamField>

<ParamField query="page_size" type="integer">
  Results per page (default: 50, max: 100). Example: `50`
</ParamField>

<Tip>
  Use `energy_classification` and `greenhouse_gaz_classification` to filter properties by energy efficiency (A = most efficient, G = least efficient).
</Tip>

<Tip>
  Check `price_has_decreased` to identify listings where prices have been reduced - useful for finding motivated sellers.
</Tip>

<Note>
  DPE/GES ratings follow French energy diagnostic standards. Properties using the July 2021 standard (`use_july_2021_energy_performance_diagnostic: true`) have more accurate ratings.
</Note>

<Tip>
  Filter by `ad_type` to separate purchases (`buy`) from rentals, and by `property_type` for specific property categories.
</Tip>

## Code Examples

<CodeGroup>
```bash cURL
curl -X GET "https://api.lobstr.io/v1/results?squid=YOUR_SQUID_HASH&run=YOUR_RUN_HASH" \
  -H "Authorization: Token YOUR_API_KEY"
```

```python Python
import requests

API_KEY = "YOUR_API_KEY"
BASE_URL = "https://api.lobstr.io/v1"
headers = {"Authorization": f"Token {API_KEY}"}

squid_id = "YOUR_SQUID_HASH"
run_id = "YOUR_RUN_HASH"

# Get Bien'Ici real estate listing results
response = requests.get(f"{BASE_URL}/results",
    headers=headers,
    params={"squid": squid_id, "run": run_id}
)

data = response.json()
print(f"Total listings: {data['total_results']}")

for listing in data["data"]:
    print(f"\n{listing['title']}")
    print(f"  Type: {listing['property_type']}")
    print(f"  Price: €{listing['price']:,}")
    print(f"  Area: {listing['surface_area']} m²")
    print(f"  Location: {listing['city']} ({listing['postal_code']})")
    print(f"  Rooms: {listing['rooms_quantity']}")
    if listing.get('energy_classification'):
        print(f"  Energy Rating (DPE): {listing['energy_classification']}")
    print(f"  URL: {listing['url']}")
```
</CodeGroup>
