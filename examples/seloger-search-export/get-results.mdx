---
title: "Get Results"
description: "Retrieve scraped data from SeLoger Search Export"
api: "GET https://api.lobstr.io/v1/results"
---

Retrieve scraped data from your **SeLoger Search Export** runs.

## Headers
<ParamField header="Authorization" type="string" required>
  Your API authentication token. Value: `Token YOUR_API_KEY`
</ParamField>

## Query Parameters
<ParamField query="squid" type="string" required>
  Hash of the squid to get results from
</ParamField>

<ParamField query="run" type="string">
  Hash of a specific run (optional)
</ParamField>

<ParamField query="page" type="integer">
  Page number (default: 1). Example: `1`
</ParamField>

## Result Fields
Each result object contains the following fields:

<ResponseField name="address" type="string">
  Full address of the property. Example: `Issy-les-Moulineaux`
</ResponseField>

<ResponseField name="is_furnished" type="boolean">
  Whether the property is furnished. Example: `false`
</ResponseField>

<ResponseField name="agency_contact_name" type="string">
  Name of the agency contact person. Example: `GEMALIA IMMOBILIER`
</ResponseField>

<ResponseField name="agency_has_email" type="boolean">
  Whether the agency has a listed email address. Example: `true`
</ResponseField>

<ResponseField name="agency_id" type="integer">
  Unique identifier for the agency on SeLoger. Example: `440227`
</ResponseField>

<ResponseField name="agency_img_url" type="string">
  URL of the agency's logo or profile image. Example: `https://v.seloger.com/s/width/150/logos/0/z/x/0/0zx09bbylr0bhqsjhjdfui8t37znwvg2nyozrq5s6.jpg`
</ResponseField>

<ResponseField name="agency_link" type="string">
  URL to the agency's profile page on SeLoger. Example: `http://www.gemalia.fr/`
</ResponseField>

<ResponseField name="agency_page" type="string">

<ResponseField name="agency_phone_number" type="string">
  Phone number of the agency. Example: `06 08 16 60 22`
