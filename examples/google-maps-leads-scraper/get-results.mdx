---
title: "Get Results"
description: "Retrieve scraped data from Google Maps Leads Scraper"
api: "GET https://api.lobstr.io/v1/results"
---

Retrieve scraped data from your **Google Maps Leads Scraper** runs.

## Headers
<ParamField header="Authorization" type="string" required>
  Your API authentication token. Value: `Token YOUR_API_KEY`
</ParamField>

## Query Parameters
<ParamField query="squid" type="string" required>
  Hash of the squid to get results from
</ParamField>

<ParamField query="run" type="string">
  Hash of a specific run (optional)
</ParamField>

<ParamField query="page" type="integer">
  Page number (default: 1). Example: `1`
</ParamField>

## Result Fields
Each result object contains the following fields:

<ResponseField name="match_filters" type="boolean">
  Whether the result passed your geo and category filters. Example: `true`
</ResponseField>

<ResponseField name="no_match_reasons" type="string">
  Reasons why the result did not match your filters
</ResponseField>

<ResponseField name="zero_x" type="string">
  Internal Google Maps hex identifier for the business. Example: `0x30e29f5331420fff:0x40714ec1d130200a`
</ResponseField>

<ResponseField name="cid" type="string">
  Google Maps unique numeric business identifier (CID). Example: `4643579285140938762`
</ResponseField>

<ResponseField name="place_id" type="string">
  Google Places API unique identifier for the business. Example: `ChIJ_w9CMVOf4jARCiAw0cFOcUA`
</ResponseField>

<ResponseField name="url" type="string">
  Direct Google Maps URL to the business listing. Example: `https://www.google.com/maps?cid=4643579285140938762`
</ResponseField>

<ResponseField name="name" type="string">
  Business name as listed on Google Maps. Example: `The Mesh Bangkok Sukhumvit 22`
</ResponseField>

<ResponseField name="address" type="string">
