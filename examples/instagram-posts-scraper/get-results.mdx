---
title: "Get Results"
description: "Retrieve scraped data from Instagram Post Scraper"
api: "GET https://api.lobstr.io/v1/results"
---

Retrieve scraped data from your **Instagram Post Scraper** runs.

## Headers
<ParamField header="Authorization" type="string" required>
  Your API authentication token. Value: `Token YOUR_API_KEY`
</ParamField>

## Query Parameters
<ParamField query="squid" type="string" required>
  Hash of the squid to get results from
</ParamField>

<ParamField query="run" type="string">
  Hash of a specific run (optional)
</ParamField>

<ParamField query="page" type="integer">
  Page number (default: 1). Example: `1`
</ParamField>

  Unique identifier for the Instagram post. Example: `3832413338916167164`
</ResponseField>

  Shortcode used in the Instagram post URL (e.g. instagram.com/p/[shortcode]). Example: `DUvd6BykR38`
