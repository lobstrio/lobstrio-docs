---
title: "Update Settings"
description: "Configure scraper settings for Vinted Products Scraper"
api: "POST https://api.lobstr.io/v1/squids/{squid_hash}"
---

Configure your squid settings for the Vinted Products Scraper. Control pagination depth to limit the number of catalog pages scraped.

## Headers

<ParamField header="Authorization" type="string" required>
  Your API authentication token. Value: `Token YOUR_API_KEY`
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Request body format. Value: `application/json`
</ParamField>

## Scraper Parameters

Set these parameters in the `params` object:

| Parameter | Type | Default | Description |
| --- | --- | --- | --- |
| **max_pages** | integer | null | Maximum number of catalog pages to scrape per task |

## Squid Settings

Configure general squid settings:

| Setting | Type | Description |
| --- | --- | --- |
| **name** | string | Display name for your squid configuration |
| **concurrency** | integer | Number of parallel scraping workers (default: 1) |
| **export_unique_results** | boolean | Export only unique results (deduplicated) |
| **to_complete** | boolean | Run until all tasks complete |
| **no_line_breaks** | boolean | Remove line breaks from results |

<Tip>
  Set `max_pages` to limit scraping depth and control costs for large catalog searches.
</Tip>

<Note>
  Vinted catalog pages typically contain 24-48 products per page.
</Note>

## Code Examples

<CodeGroup>
```bash cURL
curl -X POST "https://api.lobstr.io/v1/squids/YOUR_SQUID_HASH" \
  -H "Authorization: Token YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Vinted Products Scraper (1)",
    "concurrency": 1,
    "export_unique_results": true,
    "to_complete": false,
    "no_line_breaks": true,
    "params": {
      "max_pages": 10
    }
  }'
```

```python Python
import requests

API_KEY = "YOUR_API_KEY"
BASE_URL = "https://api.lobstr.io/v1"
headers = {"Authorization": f"Token {API_KEY}"}

squid_id = "YOUR_SQUID_HASH"

# Update squid settings with pagination limit
payload = {
    "name": "Vinted Products Scraper (1)",
    "concurrency": 1,
    "export_unique_results": True,
    "to_complete": False,
    "no_line_breaks": True,
    "params": {
        "max_pages": 10  # Limit catalog pages to scrape
    }
}

response = requests.post(f"{BASE_URL}/squids/{squid_id}",
    headers=headers,
    json=payload
)

print(f"Settings updated: {response.json()['name']}")
```
</CodeGroup>
